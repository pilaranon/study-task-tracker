# Study Planner / Task Tracker

## Description
A cloud-based task management app that allows users to create, edit, delete, and manage tasks with priorities and due dates.

## Features
- User authentication
- Task CRUD operations
- Task sorting
- Mark tasks complete
- Dashboard UI

## Tech Stack
- Flask (Backend)
- HTML/CSS/JS (Frontend)
- SQLite (Development)
- AWS RDS (Production)
- Docker + EC2 (Deployment)

## Setup
```bash
pip install -r requirements.txt
python app.py
